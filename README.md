### About Me

I'm a **DevOps / SRE engineer with 5+ years of experience** building and operating cloud-native infrastructure across AWS, GCP, and Azure. I've managed **100+ Kubernetes clusters**, worked across **multi-cloud environments**, and supported platform teams running **10–50 microservices** in production.

My day-to-day spans designing CI/CD strategy, managing cloud infrastructure, writing IaC, and owning observability — from building pipelines from scratch to responding to incidents and improving reliability through SLOs and error budgets.

I care deeply about reducing toil, keeping systems observable, and building infra that teams can trust.

- 🔧 Currently working on: hardening Kubernetes cluster operations & preparing for CKA
- 🌍 Multi-cloud: AWS · GCP · Azure — all three in production
- 📉 SRE practices: SLOs, error budgets, runbooks, blameless postmortems
- 🔁 CI/CD: designing org-wide pipeline standards across teams
